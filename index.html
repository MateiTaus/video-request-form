Solicitare video

Formular simplu, gandit pentru persoane non-tehnice. Daca lipseste ceva obligatoriu, vezi mesajul rosu sub camp.

Detalii generale
Camp optional.
Ratio se aplica selectiei active.
Generation Type *
Setari: Text to Video
Setari: Image to Video
Start frame reprezinta cum va incepe videoul, last frame reprezinta cum se va sfarsi videoul. Pentru selectia "Image to video" este obligatorie introducerea start frame-ului. Daca nu se introduce last frame videoul se va termina conform promptului.
Accepta: jpg, png, webp.
Optional.
Setari: Reference to Video
Pentru aceasta selectie, se pot incarca maxim 3 imagini de referinta, ele nu reprezinta start sau last frame, sunt strict pentru referinta si se vor folosi in concordanta cu promptul.
* = obligatoriu. Formularul nu se trimite pana nu e complet corect.
